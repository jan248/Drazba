# Drazba
Aplikacija za virtualno dražbo starinskih in za nekatere neprecenljivih predmetov. Naredil sem spletno stran na katero se registrirate in na njej sodelujete pri dražbah različnih predmetov. Za registracijo je potrebno vnesti podatke ime, priimek in uporabniško ime ter geslo, ki ga je potrebno potrditi.
